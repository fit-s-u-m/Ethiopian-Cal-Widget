## 🚀 Installation

1. Clone or download this repository:

   ```bash
   git clone https://github.com/fit-s-u-m/Ethiopian-Cal-Widget.git
   ```

2. Move the widget to your Noctalia widgets directory:

   ```bash
   mv ethiopian-calendar ~/.config/noctalia/widgets/
   ```

3. Restart Noctalia Shell or reload widgets:

   ```bash
   killall qs
   ```
   ```bash
   qs -c noctalia-shell
   ```
---

## 🧠 About the Ethiopian Calendar

The Ethiopian calendar consists of:

* **13 months** (12 months of 30 days + 1 short month)
* A unique and rich historical system
* A calendar that is approximately **7–8 years behind** the Gregorian calendar
